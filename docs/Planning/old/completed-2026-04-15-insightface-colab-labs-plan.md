# Archived Plan Note

The previously active plan for the InsightFace Labs rollout via Google Colab has been superseded by the backend-first model runtime transition plan created on `2026-04-16`.

The original active plan file remains in `docs/Planning/` because the workspace denied a physical move at archive time, but it should be treated as inactive and historical.

Use the newer active plan in:

- `docs/Planning/2026-04-16-backend-model-runtime-transition-plan.md`

Reason for archive:

- Google Colab is no longer the preferred long-term hosting path for InsightFace inference in this repo.
- The new direction is to run model inference directly in the backend while preserving a plug-and-play model architecture that can support new or multiple embedding models later.
